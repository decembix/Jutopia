# Jutopia

> 대학 동아리 운영을 하나로 연결하는 통합 플랫폼

---

# 🎓 프로젝트 소개

대학 동아리 운영은 보통 여러 플랫폼에 분산되어 이루어집니다.

- 공지는 카카오톡
- 일정 관리는 Google Calendar
- 회의록과 자료는 Google Drive
- 모집은 Google Forms
- 활동 기록은 Notion
- 회비 관리는 Excel

이처럼 정보와 기능이 흩어져 있으면  
운영진과 부원 모두 관리와 접근에 어려움을 겪게 됩니다.

**Jutopia**는 이러한 문제를 해결하기 위해 제작된  
대학생 동아리 통합 운영 플랫폼입니다.

사용자는 하나의 서비스 안에서:

- 여러 동아리 활동을 동시에 관리하고,
- 동아리 내부의 다양한 모임에 참여하며,
- 일정·공지·출석·활동 정보를 통합적으로 확인할 수 있습니다.

---

# ✨ 핵심 특징

## 1. 다중 동아리 구조 지원

기존 단일 동아리 중심 서비스와 달리,  
Jutopia는 사용자가 여러 동아리에 동시에 가입할 수 있도록 설계되었습니다.

예시:

- 학술 동아리
- 공연 동아리
- 프로젝트 팀
- 스터디 모임

등 다양한 활동을 하나의 계정에서 관리할 수 있습니다.

---

## 2. 다중 모임 구조 지원

각 동아리 내부에는 여러 소모임과 팀이 존재할 수 있습니다.

예시:

- 공연팀
- 세션팀
- 스터디팀
- 프로젝트팀
- 운영팀

사용자는 자신이 속한 모든 모임을 통합적으로 확인할 수 있습니다.

---

## 3. 통합 캘린더

사용자가 속한 모든 동아리와 모임의 일정이  
하나의 캘린더에 통합되어 표시됩니다.

예시:

- 정기 모임
- 공연 일정
- 스터디 일정
- 회의
- MT
- 행사 일정

동아리별 색상 및 태그를 통해 쉽게 구분할 수 있습니다.

---

## 4. 동아리 코드 기반 가입 시스템

운영진이 생성한 동아리 코드를 통해  
사용자는 원하는 동아리에 쉽게 참가할 수 있습니다.

또한 동아리별로:

- 기수
- 역할
- 활동 상태
- 파트

등 추가 정보를 별도로 관리할 수 있습니다.

---

## 5. 동아리 맞춤형 운영 구조

동아리마다 서로 다른 운영 방식을 반영할 수 있도록 설계되었습니다.

운영진은:

- 역할 구조 설정
- 소개 문구 수정
- 대표 이미지 설정
- 메인 화면 커스터마이징

등을 자유롭게 변경할 수 있습니다.
